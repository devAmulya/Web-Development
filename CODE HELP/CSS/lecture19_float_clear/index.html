1) Position Property: defines how to position an element
2) static(default) - No top, right , bottom , left property and no z index applied
3) relative - positioning ,using (top, right, etc) space property, relative to current window/device # gap is maintained
4) absolute - For overlapping absolute positioning using space property relative to parent class(till relative element found or html) # gap is not maintained - element space is removed
5) fixed - element remain intact to its position in windows even if scrolls up/down irrespective of its parent container(entire web page) space property can be used
6) sticky - element remain intact to its positon in windows even if scrolls up/down till its present container space property can be used
7) Good practise - keep parent container relative
8) z-index : defines konsa upr ayega ek element jyada z, towards user(upr) , lesser z towards screen(niche) in css, usually child covers(above) the parents class
